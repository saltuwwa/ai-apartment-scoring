PostgreSQL — каталог мебели

БД хранит каталог мебели с ценами. После стейджинга можно сопоставить добавленную мебель с реальными ценами.

Локальный запуск БД

# Установить PostgreSQL, создать БД
createdb real_estate
psql -d real_estate -f init_furniture_db.sql

# В .env добавить:
POSTGRES_HOST=localhost
POSTGRES_PORT=5432
POSTGRES_DB=real_estate
POSTGRES_USER=postgres
POSTGRES_PASSWORD=ваш_пароль

Облачное развёртывание

Локально: python run_server.py — сайт на http://localhost:8000

Docker:

docker build -t realestate-app .
docker run -p 8000:8000 --env-file .env realestate-app

Облако (Railway, Render, Fly.io): загрузите проект, добавьте переменные GEMINI_API_KEY, OPENAI_API_KEY. Для БД используйте облачный PostgreSQL (Neon, Supabase, Railway).
